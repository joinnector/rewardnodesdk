# Node SDK
nector.io is a B2B SaaS loyalty platform that allows the business to target a wider audience and increase customer engagement by rewarding them.

## Dependency
NodeJS 12.18.1

## Installation
- `npm install joinnector` 
	or 
- `yarn add joinnector`
