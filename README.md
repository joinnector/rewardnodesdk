# Node SDK
nector.io is a loyalty and reward APIs platform, that allows businesses to reward deals, offers, wallet points to their customers for various actions.

## Dependency
NodeJS 12.18.1

## Installation
- `npm install joinnector` 
	or 
- `yarn add joinnector`
